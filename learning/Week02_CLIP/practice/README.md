# Week02 Practice Overview

`practice/` 是本週所有練習的唯一入口。

- [Concept Practice](./concept/concept_practice.md)：確認 CLIP、共同嵌入空間、相似度與零樣本分類觀念。
- [Concept Answer Key](./concept/concept_answer_key.md)：完成作答後再查看的參考說明。
- [Coding Practice](./coding/coding_practice.md)：執行 Guided Demo 並記錄 shape、中間值、錯誤與理解。
- [Coding Observation Key](./coding/coding_observation_key.md)：提供觀察方向，不是唯一標準答案。

本週採 Guided Code Reading Mode（引導式程式閱讀模式）。`guided_demos/` 中的程式完整可執行，不要求從零實作、不使用 TODO，也不建立 `exercises/` 或 `solutions/`。

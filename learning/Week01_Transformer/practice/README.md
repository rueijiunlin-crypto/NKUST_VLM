# Week01 Practice Overview（練習總覽）

本資料夾是 Week01 Transformer 基礎的練習總入口。

## 練習順序

1. [Concept Practice](./concept/concept_practice.md)
2. [Guided Code Reading Overview](./coding/README.md)
3. [Coding Observation Tasks](./coding/coding_practice.md)
4. 執行 [`coding/guided_demos/`](./coding/guided_demos/)。
5. 參考 [Coding Observation Key](./coding/coding_observation_key.md) 檢查觀察方向。

## 練習內容

- `concept/`：觀念練習與觀念參考答案。
- `coding/`：Guided Code Reading 程式、觀察任務、觀察指引與套件需求。

## 學習模式

Week01 採用 Guided Code Reading Mode（引導式程式閱讀模式）：

- 不要求從零實作 Transformer。
- 不使用 `exercises/`、`solutions/` 或 TODO 補空題。
- 透過完整可執行程式觀察 shape、中間值與資料流。
- 可修改小參數，記錄輸出如何改變。

[Concept Answer Key（觀念參考答案）](./concept/concept_answer_key.md) 請完成觀念練習後再查看。`coding_observation_key.md` 是觀察指引，不是標準答案。

## 環境需求

程式練習使用 Python（程式語言）與 NumPy（數值運算函式庫）。請參考 [`coding/requirements.txt`](./coding/requirements.txt)。

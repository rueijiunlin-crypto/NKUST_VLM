# Week01 Practice Overview（練習總覽）

本資料夾是 Week01 Transformer 基礎的練習總入口。

## 練習順序

1. [Concept Practice](./concept/concept_practice.md)
2. [Coding Practice](./coding/coding_practice.md)
3. 完成後再查看參考答案與參考程式。

## 練習內容

- `concept/`：觀念練習與觀念參考答案。
- `coding/`：程式練習索引、練習檔、參考程式與套件需求。

## 查看參考答案提醒

- [Concept Answer Key（觀念參考答案）](./concept/concept_answer_key.md) 請完成觀念練習後再查看。
- [Coding Answer Key（程式參考說明）](./coding/coding_answer_key.md) 請完成 `coding/exercises/` 後再查看。
- [Coding Solutions（程式參考解答）](./coding/solutions/) 是完整可執行參考程式，不應直接複製到 `exercises/`。

## 環境需求

程式練習使用 Python（程式語言）與 NumPy（數值運算函式庫）。請參考 [`coding/requirements.txt`](./coding/requirements.txt)。

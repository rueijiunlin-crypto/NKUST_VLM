# AGENTS.md

## 專案原則

- 本專案所有文件預設使用繁體中文撰寫。
- 英文專有名詞第一次出現時，需附上中文翻譯，例如 Vision-Language Model（視覺語言模型）。
- 所有文件以 Vision-Language Model（視覺語言模型）與 Vision-Language-Action Model（視覺語言動作模型）的碩士研究為核心。
- 不得新增船體、浮力、USV、海浪、PINNs 相關內容；若需要提及，只能出現在排除項目或限制說明中。
- 所有資料夾與文件需適合後續以一週一週的方式學習、實作、驗收與回顧。
- 文件應清楚、可維護，並能作為 Notion（筆記與資料庫管理工具）建置依據。

## 文件撰寫規範

- 優先說明學習目標、必懂概念、實作任務、驗收標準與輸出成果。
- 程式與技術資料夾可使用英文命名，但 README 內容需以繁體中文說明。
- 每週資料夾需能對應 Notion 的 Learning Roadmap Database（學習路線資料庫）。
- 實驗紀錄需保留目的、環境、步驟、結果、問題、修正方式與下一步。
